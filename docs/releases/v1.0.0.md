## Recommended Deployment Procedure

1. Build the image from the repository root.
2. Publish the image to the target container registry.
3. Deploy the new image to Cloud Run with the required environment variables.
4. Confirm the Telegram webhook points to the final `/webhook` URL.

Reference operational guide: `docs/operations.md`.

## Post-Deploy Smoke Test

1. Check that the Cloud Run service starts and responds to HTTP requests.
2. Send a PDF document to the Telegram bot.
3. Confirm the webhook receives the update successfully.
4. Confirm the destination mailbox receives the email with the expected attachment.
5. Review Cloud Run logs for authentication or delivery failures.
